# TripGenie AI ✈️

An AI-powered travel planning application built using React and Tailwind CSS.

## Features
- Create personalized trips
- Budget-based planning
- Interest-based recommendations
- Dynamic itinerary generation
- Beautiful responsive UI

## Tech Stack
- React.js
- React Router
- Tailwind CSS
- Local Storage

## Future Improvements
- Gemini AI integration
- Hotel recommendations
- Destination images
- Real-time travel suggestions
- User authentication
